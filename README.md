# MUMBAI-CLIMATE-
Problem Statement

Analyze Mumbai's daily climate data to identify weather trends, seasonal patterns, and key environmental insights through interactive visualizations.

Solution Explanation

Developed an interactive Power BI dashboard that transforms historical climate data into meaningful visual insights using KPIs, charts, filters, and trend analysis.

Future Scope

Enhance the project by integrating real-time weather APIs, predictive analytics, and machine learning models for climate forecasting and decision support.
